O VSS-Vision é um sistema de visão computacional open-source que faz parte do VSS-SDK.
Mais informações sobre o framework podem ser encontradas em [VSS-SDK][vss-sdk].


## Installation
```
$ sudo ./configure.sh
$ vss-vision
```

## Development
First Time
```
$ sudo ./configure_with_tests.sh
```

Tests
```
$ cd build
$ make
$ ./tests
```

Execution
```
$ cd build
$ make
$ ./vss-vision
```
